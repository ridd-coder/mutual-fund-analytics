# Mutual Fund Analytics

## Project Overview

This project analyzes Indian mutual fund data using Python, Pandas, and financial APIs.

## Day 1 Deliverables

* Project structure created
* Dependencies installed
* NAV data fetched using mfapi.in
* 10 CSV datasets generated and stored
* Data ingestion completed
* Data quality validation completed
* AMFI scheme code validation completed

## Technologies

* Python
* Pandas
* NumPy
* Requests
* Plotly
* SQLAlchemy
* Jupyter Notebook
